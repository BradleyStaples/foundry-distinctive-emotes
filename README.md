# foundry-distinctive-emotes

Distinctive emotes styling for Foundry VTT

A simple module to change the appearance of emotes to make them more distinctive from other chat message entries. The changes made are:

- hide the character name, player name, and timestamp
- change the text to be in a customizable color
- add a dotted border to the left of the text in a customizable color
- add some spacing around the text

To customize the text and border colors, go to `Settings > Game Settings > Distinctive Emotes`.

### Installation Instructions

To install a module, follow these instructions:

1. Start FVTT and browse to the Game Modules tab in the Configuration and Setup menu
2. Select the Install Module button and enter the following URL: https://raw.githubusercontent.com/BradleyStaples/foundry-distinctive-emotes/master/module.json
3. Click Install and wait for installation to complete
